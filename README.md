## 拼图游戏商城（PSD设计稿）

#### 一：项目描述

​	这是一个标志着PS网页设计这个课程结束的大项目。

​	要求自我设计，独立制作一个“在线商城设计稿”。

​	要求自主设计Logo，网站要包含首页，商品详情页，购物车页，个人信息页。

​	页面宽度为1440px，版心自定，但不能脱离主流尺寸！
